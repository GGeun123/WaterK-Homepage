## 주제
### 회사 홈페이지

## 팀원
### 송종근, 김채연

## 작품 개요 
- 회사에서 사용하는 CRUD 홈페이지를 제작
- 스크롤, 채팅, 회원가입 등등의 기능을 이용하여 편의성 제공

## 제작기간
- 2022.09.20 ~ 2022.09.30

## 사용 기술
: PHP, HTML, CSS, Javascript, Jquery, MSSQL
